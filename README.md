# 🧠 What this app does?

This app takes input in seconds and converts it into a structured time format.

### Example:
Input: 90061 seconds  
Output:
1 Day  
1 Hour  
1 Minute  
1 Second  

---

# ⚙️ Features

- ⏱ Converts seconds into days, hours, minutes, seconds
- 📌 Input validation (empty input handling)

# 🚀 How to Run

### 1️⃣ Clone the repository
```bash
git clone https://github.com/ashish-modak-22/Kotlin_TimeConverterApp.git
```

### 2️⃣ Open in Android Studio
- Open Android Studio
- Click on **Open Project**
- Select the cloned folder

### 3️⃣ Run the app
- Click ▶ Run button
- Or use emulator / physical device
